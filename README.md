# Nexus AI Agency - Landing Page

![Live Status](https://img.shields.io/badge/Status-Live-brightgreen)
![Vercel](https://img.shields.io/badge/Deployed_on-Vercel-black?logo=vercel)

A modern, high-performance landing page for a fictional AI Creative Agency, featuring a sleek dark mode UI with glassmorphism effects and dynamic content grids.

**🌐 Live Demo:** [https://inamigos-task3-five.vercel.app/](https://inamigos-task3-five.vercel.app/)

## 🚀 Features

- **Glassmorphism Design:** Beautiful semi-transparent cards with blur effects.
- **Fully Responsive:** CSS Grid and Flexbox layouts that adapt seamlessly to mobile, tablet, and desktop screens.
- **Dynamic CSS Animations:** Floating background blobs and smooth hover transitions.
- **Comprehensive Sections:**
  - Hero with animated background
  - About Us & Company Stats
  - Services Grid (AI Integration, Cloud Architecture, etc.)
  - Features (Why Choose Us)
  - Client Testimonials
  - 3-Tier Pricing Table
  - Contact Form

## 🛠️ Technologies Used

- **HTML5:** Semantic structure and layout.
- **CSS3 (Vanilla):** Custom styling, variables, flexbox/grid, and animations without relying on external frameworks.
- **JavaScript:** (Prepared for future interactivity/form handling).
- **Google Fonts:** Utilizing the modern 'Outfit' font family.

## 💻 Local Development

To run this project locally, simply clone the repository and open the `index.html` file in your preferred web browser. No build steps or package installations are required!

```bash
git clone https://github.com/AliveCore729/Inamigos-Task3.git
cd Inamigos-Task3
# Open index.html in a browser
```

## 📦 Deployment

This project is automatically deployed via [Vercel](https://vercel.com/). Any pushes to the `main` branch trigger a new production build.
